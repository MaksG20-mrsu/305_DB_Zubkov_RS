# Task01

## Структура каталога

В каталоге `Task01` находятся файлы исходных данных и результаты выполнения задания.

### Исходные данные
- `movies.csv` — список фильмов.
- `ratings.csv` — оценки пользователей для фильмов.
- `tags.csv` — пользовательские теги к фильмам.
- `users.txt` — информация о пользователях.
- `genres.txt` — список жанров.
- `occupation.txt` — список профессий пользователей.

### Результаты выполнения задания
- `ratings_count.txt` — минимальный и максимальный идентификатор пользователя из файла `ratings.csv` и количество строк с этими идентификаторами.
- `sqlite.txt` — версия установленной SQLite и режимы вывода данных, доступные в утилите `sqlite3`.
